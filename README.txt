   images/hero.jpg

3. Project images:
   images/work1.jpg
   images/work2.jpg
   images/work3.jpg
   images/work4.jpg
   images/work5.jpg
   images/work6.jpg

4. Certificate images:
   images/certificate1.jpg
   images/certificate2.jpg
   images/certificate3.jpg
   images/certificate4.jpg
   images/certificate5.jpg
   images/certificate6.jpg

IMPORTANT FILE-NAMING RULES
- Do not use spaces in filenames.
- Do not write certificate1.jpg.jpg
- Do not write certificate 1.jpg
- Correct name: certificate1.jpg
- Folder name must be images, not image.
- GitHub filenames are case-sensitive.
- certificate1.jpg and Certificate1.jpg are different names.

RECOMMENDED CERTIFICATE IMAGE SIZE
- Landscape certificate: 1600 × 1100 px or similar
- Keep all certificates in JPG format for easy loading.
- Compress each image before uploading.
- Try to keep each file under 500 KB to 1 MB.
